Oi! Start navigating through Raviolist with the left sidebar!

Raviolist is a collaborative project about presenting the move lists of all the characters from as many fighting games as possible. It's a tool created for the Fighting Game Community.

Raviolist is a web app, you can add it to your mobile device home screen and it will act like a native application would. It requires no permission and will always stay up-to-date. Great!
